== TravelSMS

Created in 36 hours while at MHacks VI 2015. 

SMS-based travel assistance application built using Twilio. Hosted on Heroku.

To use: Send a text message containing location (e.g. 'San Francisco' or 'University of Texas at Austin') to (832) 742-1996.

APIs:
* Yelp
* Forecast.io
* Geocoder
